Import von Fremdformaten

Über das Menü "Datei/Importieren" können folgende Dateiformate importiert werden:

  • CLONE Format
  • GenBank Format
  • Sequenzierungsdaten (ABI/Chromas Format)
  • FASTA Format
  • "reine" DNA- oder Aminosäure-Daten ("plain text")
Die Standardeinstellung "Automatisch" kann dabei gängige Formate erkennen. Falls dies fehlschlägt, muss das Format manuell angegeben werden.

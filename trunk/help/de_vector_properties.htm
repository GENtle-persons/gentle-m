Vektoreigenschaften, Features und Restriktionsenzyme bearbeiten

Der Dialog zum Bearbeiten von Vektoreigenschaften, Features und Restriktionsenzymen kann über Kontextmenüs in der Plasmid- und Sequenzkarte sowie über einen Doppelklick auf den "leeren" Plasmidhintergrund aufgerufen werden.

Vektoreigenschaften bearbeiten

Hier können der Vektorname, Beschreibung, sowie die "sticky ends" bearbeitet werden.

Features bearbeiten

Hier hönnen Name, Position, Länge, Typ, Beschreibung, Farbe und ggfs. Leseraster eines Features bearbeitet werden. Ebenso können Features gelöscht oder neue hinzugefügt werden. Ein Doppelklick auf ein Feature in der Plasmidkarte startet diesen Dialog und wählt das Feature zum Bearbeiten aus.

Restriktionsenzyme bearbeiten

Hier können Restriktionsenzyme zur aktuellen Sequenz-Darstellung hinzugefügt oder daraus entfernt werden. Die linke Liste zeigt dabei die aktuell verwendeten Enzyme an. Rechts oben stehen frei definierbare Gruppen von Enzymen. Die in einer Gruppe enthaltenen enzyme werden durch anwählen der Gruppe rechts unten angezeigt.

Es können einzelne Enzyme aus dieser Liste, oder alle Enzyme einer Gruppe hinzugefügt werden.

Um eine neue Gruppe von Enzymen zu erstellen, oder um Enzyme zu einer bestehenden Gruppe hinzuzufüren, müssen diese Enzyme in der linken Liste ausgewählt werden.

Zum Bearbeiten von Enzymgruppen ohne geöffneten Vektor kann dieser Dialog auch über das Menü "Werkzeuge/Enzym-Editor" aufgerufen werden.

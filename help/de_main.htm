GENtle Hilfe

GENtle ist ein Programm zur Bearbeitung und Verwaltung von DNA- und Aminosäuresequenzen.

Achtung : Diese Hilfe-Datei ist nicht mehr aktuell. Bitte verwenden Sie die Online-Hilfe hier!

  • Tastatur- und Mauskürzel
  • Laden und Speichern in der Datenbank
  • Datenbank verwalten
  • Import von Fremdformaten
  • Direkteingabe von Sequenzen
  • Vektoren/Sequenzen/Primer bearbeiten
  • Vektoreigenschaften, Features und Restriktionsenzyme bearbeiten
  • Restriktionsverdau
  • Ligation
  • PCR und Primer-Design
  • Alignments
  • Aminosäuresequenzen

Diese Software steht unter der GNU general public license (GPL) jedem frei zur Verfügung. Diese Software verwendet SQLite (public domain) als Datenbank, wxWindows (open source) als GUI, sowie blastcl3 (NBCI). Der Installer ist Nullsoft Scriptable Install System (open source).

GENtle help

GENtle is a program for mainpulating and organizing DNA and amino acid sequences.
  • Keyboard and mouse shortcuts
  • Opening and storing files in the database
  • Database management
  • Importing other file formats
  • Enter sequences
  • Editing vectors/sequences/primers
  • Editing properties of vectors, features and enzymes
  • DNA restriction
  • Ligation
  • PCR and primer design
  • Alignments
  • Amino acid sequences

This software is released under GNU general public license (GPL). This software uses SQLite (public domain) as its database, wxWindows (open source) as GUI, and blastcl3 (NBCI). The installer was created by the Nullsoft Scriptable Install System (open source).

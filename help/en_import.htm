Import of other file formats

Through the menu "File/Import", files in other formats can be imported:

  • CLONE format
  • GenBank format
  • Sequencer data (ABI/Chromas format)
  • FASTA format
  • "plain text" DNA and amino acid sequences
The default "automatic" setting can recognize some of these formats. If that fails, the file type has to be selected manually.
